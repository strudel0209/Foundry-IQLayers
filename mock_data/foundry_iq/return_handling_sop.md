# SOP-RET-001: Product Return Handling Procedure
**Effective Date:** 2025-09-01 | **Review Date:** 2026-09-01
**Department:** Operations | **Owner:** VP Operations

## 1. Purpose
Defines the standard procedure for processing product returns at all Lakeshore Retail locations.

## 2. Scope
Applies to all store managers, shift supervisors, and customer service representatives.

## 3. Return Acceptance Criteria
- Returns accepted within 30 days of purchase with valid receipt
- Product must be in original packaging (exceptions for quality defects)
- Perishable items: returns accepted within 48 hours for quality issues

## 4. Quality Defect Returns
When a customer reports a quality defect:
1. Document the defect type (texture, taste, appearance, contamination)
2. Photograph the product
3. Log the return in the POS system with reason code "Quality-*"
4. Offer full refund plus a replacement product
5. Place the returned product in the quality hold freezer

## 5. Escalation Triggers
**Automatic escalation to Quality Assurance team when:**
- Same product receives >5 quality returns in a single store within 7 days
- Same product receives >10 quality returns across all stores within 14 days
- Any contamination report (immediate escalation)

## 6. Escalation Process
1. Store manager emails quality@lakeshoreretail.com with return details
2. QA team initiates batch investigation within 24 hours
3. Supplier notification sent within 48 hours if batch defect confirmed
4. Product hold/pull decision made by QA Director within 72 hours
