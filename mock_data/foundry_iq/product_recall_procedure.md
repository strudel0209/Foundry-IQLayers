# SOP-QA-007: Product Recall Procedure
**Effective Date:** 2025-03-01 | **Department:** Quality Assurance

## 1. Recall Classification
- **Class I**: Reasonable probability of health consequences
- **Class II**: Possible but unlikely health consequences
- **Class III**: Quality issue only, no health impact

## 2. Decision Authority
- Class I: CEO + General Counsel (immediate)
- Class II: VP Operations + QA Director (within 24 hours)
- Class III: QA Director (within 72 hours)

## 3. Recall Process
1. Stop all distribution of affected product
2. Notify all stores to pull product from shelves
3. Contact supplier for batch-level traceability
4. Prepare customer notification (if Class I or II)
5. Set up return/refund process
6. File regulatory notifications if required

## 4. Record Keeping
Maintain recall records for 7 years minimum.
